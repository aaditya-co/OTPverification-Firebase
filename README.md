## Installation

Install OTPverification-Firebase.


```bash
 git clone https://github.com/adityaraj3644/OTPverification-Firebase.git
```
```bash
 cd OTPverification-Firebase
```
```bash
 flutter pub get
```
```bash
 flutter run 
```
